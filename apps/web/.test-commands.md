# Quick Test Commands Reference

## Setup (One-Time)

```bash
cd apps/web
bun add -d vitest @vitest/ui @types/bun
```

Add to `package.json`:
```json
{
  "scripts": {
    "test": "vitest",
    "test:watch": "vitest --watch",
    "test:ui": "vitest --ui",
    "test:coverage": "vitest --coverage"
  }
}
```

## Run Tests

```bash
# All tests (one-shot)
bun run test

# Watch mode (re-run on changes)
bun run test:watch

# Interactive UI
bun run test:ui

# Coverage report
bun run test:coverage
```

## Run Specific Tests

```bash
# Single file
bun run test process-result.test.ts
bun run test process-article.test.ts

# Matching pattern
bun run test -t "article state machine"
bun run test -t "error handling"

# Watch specific file
bun run test:watch process-article.test.ts
```

## Useful Flags

```bash
# Run tests in parallel
bun run test --no-threads=false

# Show test output (console.log)
bun run test --reporter=verbose

# Fail fast (stop on first failure)
bun run test --bail

# Update snapshots (if using snapshot tests)
bun run test -u
```

## CI/CD

```bash
# Run all tests + coverage (for CI)
bun run test && bun run test:coverage
```
